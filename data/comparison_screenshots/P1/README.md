# Pair 1 screenshots

The original side-by-side comparison screenshots for Pair 1 were not available when this artifact was finalized. This folder therefore includes substitute screenshots of the generated Pair 1 A/B triple and ontology graph views. They are provided to make the Pair 1 visual artifacts complete enough for inspection alongside the JSON and Turtle files.

Files:

- `P1_triple_A_view.png`: Pair 1 baseline triple graph view.
- `P1_triple_B_view.png`: Pair 1 revised triple graph view.
- `P1_ontology_A_view.png`: Pair 1 baseline ontology-compatible graph view.
- `P1_ontology_B_view.png`: Pair 1 revised ontology-compatible graph view.
