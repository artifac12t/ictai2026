The system shall allow a student organization officer to submit a room reservation request for a campus event. The request shall include the event title, requested room, expected number of attendees, and event start and end time. If the requested event time is outside normal building hours or the expected attendance is greater than the room capacity, the system shall route the request to the department coordinator for approval before confirming the reservation. The system shall send the email confirmation only after the room is available and the required approval has been granted.
