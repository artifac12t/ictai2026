# Data Dictionary

## Key result files

- `results/adjudicated/reference_changes_all.csv`: human-verified requirement changes and detection labels for triple and ontology comparison.
- `results/adjudicated/raw_triple_audit_all.csv`: representative generated or missing structures with correctness, completeness, and curation-action labels.
- `results/adjudicated/curated_corrections_all.csv`: proposed curated structures and final decisions.
- `results/summary/reference_change_detection_summary.csv`: per-pair detected/partial/missed counts.
- `results/summary/raw_triple_audit_summary.csv`: per-pair raw audit counts.
- `review/evaluator_agreement_summary.csv`: anonymized reviewer agreement rates.
- `review/adjudicated_exceptions.csv`: accepted/resolved evaluator exceptions.
